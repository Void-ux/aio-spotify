Welcome to aio-spotify
======================

aio-spotify is a modern, simple and async-ready API wrapper for Spotify.

**Features:**

- Modern Pythonic API using ``async``\/``await`` syntax
- An easy to use object oriented design
- Global rate limit handling
- Optional automatic token refreshing
